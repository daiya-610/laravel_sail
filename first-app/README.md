# Laravel sailを使用したLaravelの開発環境の構築方法

## sec01 laravel sail 起動
1\. プロジェクト作成
- Dockerを事前に起動させておく。
```
cd ~/Desktop
mkdir Laravel
cd Laravel
curl -s "https://laravel.build/first-app" | bash
```
- first-app：任意のアプリ名でOK

2\. Laravel sailを起動する
```
cd first-app
./vendor/bin/sail up -d
```
- 時間がかかるが、完了後
http://localhost/
へアクセスする。

3\. Laravel sailを停止する
```
./vendor/bin/sail stop
```
